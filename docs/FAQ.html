We will be getting a python environment set up on everyone's computer, as well as making everyone a free CoCalc. We will then be giving a course overview and go over some basics of file manipulation.

What will I get out of this course?

This course will give you a first introduction to programming in python. Whether you have learned other programming languages before or this is your first time, we have something here for you! By the end of this course you will have developed key algorthim writing skills, and introductory proficiency in the Python language.
